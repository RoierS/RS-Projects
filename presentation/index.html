Introducing React Hook Form

React Hook Form is a lightweight and powerful library for managing forms in React applications

Advantages of React Hook Form:

  • Lightweight
  • Minimal boilerplate code
  • Performance
  • Custom Hooks
  • Customizable validation and error handling

Setting Up React Hook Form

  1. Install the package: 
    
							npm install react-hook-form
  2. Import the required functions: 
    
								import { useForm } from "react-hook-form";

Form Validation with Yup

Yup is a JavaScript library for data validation. It provides a declarative way to define validation rules for your form data

Key Features of Yup:
  • Integration with React Hook Form
  • Declarative Syntax
  • Schema-based Validation
  • Custom Validation
  • Asynchronous Validation

Yup Schema Validation Methods

Validation numeric input:


						const schema = yup.object({
							age: yup.number()
									.positive()
									.integer()
									.required('Age is required'),
						});

Ensures that the input value is a string


						const schema = yup.object({
							name: yup.string().required('Name is required'),
						});
